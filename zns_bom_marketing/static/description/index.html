Method 1: Git Clone (Recommended)

🎯 Birthday Campaign Workflow

Create Template

Design birthday message template in BOM ZNS: "🎂 Happy Birthday {customer_name}! You're now {age} years old!"

Setup Campaign

Create birthday campaign, select template, configure send time (9:00 AM), set status to "Running"

Automatic Detection

System automatically finds contacts with birthdays every day at 6:00 AM

Smart Filtering

Validates phone numbers, checks opt-out status, prevents duplicate messages

Message Sending

Sends personalized ZNS messages via BOM ZNS Simple at configured time

Analytics Tracking

Tracks delivery status, updates analytics, provides performance insights
